# OPTICA

Mission optical pre-design system for early-stage optical payload definition.

## What is this?

OPTICA is a concept-stage engineering tool for exploring and comparing optical system configurations before detailed design.

It helps you:
- define mission context and observation scenario
- select sensor and detector assumptions
- estimate key first-order optical parameters
- compare candidate architectures
- review feasibility, risks, and trade-offs

## Why it matters

Early optical design decisions strongly affect performance, complexity, and cost.

OPTICA is intended to support fast pre-design reasoning before moving into detailed modelling tools such as Zemax or more advanced simulation workflows.

## Use cases

- mission and payload concept studies
- EO/IR system pre-sizing
- architecture down-selection
- first-pass feasibility assessment
- engineering discussion support

## Status

Prototype / concept-stage decision support tool.
